#图片

##调用方式

```html
    <ui-img url="http://img.png" size="2" title="图片的标题"></ui-img>
```

##参数

props       |  type    | default  |  说明
------------|----------|----------|----------
url         | String   |          |  图片url地址
size        | String   | ''       |  同时设置宽高尺寸,单位rem
title       | String   | ''       |  一行标题,浮动在图片底部
